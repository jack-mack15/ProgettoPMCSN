# ProgettoPMCSN

Nella repository è possibile trovare tutti i file csv degli esperimenti. 

Nella cartella pythonScripts è possibile trovare tutti i file python per la generazione dei grafici e il codice per la verifica modello SCALING.

transitorio: mostra i dati del transitorio. Poichè sono sampling temporali utili solo al transitorio, non sono salvati tutti quelli usati.

BatchResults_base: dati del modello base (obiettivo 1).

BatchResults_f2a: dati del modello F2A, quindi i dati usati nei grafici che affiancano modello base e modello F2A.

BatchResults_copies: dati che mostrano le performance del modello SCALING con il parametro C che varia da 1 a 15.

BatchResults_scaling_120: dati che mostrano le performance (solo server B) del modello SCALING con C che varia da 1 a 120.

CopyMeans_report: dati che mostrano numero copie medie server B in funzione del parametro C.

BatchMeans_scal_finale: contiene i dati del modello SCALING che vengono affiancati nei grafici dal modello F2A (ultimi grafici del report).
